# 🩺 Bhaskar's Personal Health Tracker

An interactive, full-stack health logging application built with **C++, HTML/CSS, and JavaScript**, designed to track daily wellness metrics like sleep, steps walked, and hydration levels. The project combines strong backend logic with a clean, animated frontend to deliver both functionality and flair.

---

## 💡 Features

- ✅ Input tracking for **Sleep Hours**, **Steps Walked**, and **Water Intake**
- 🧠 Intelligent backend analytics for trend detection using **C++**
- 🌐 Responsive frontend built with **HTML5, CSS3, and vanilla JS**
- 📊 Live preview sidebar showing recent health logs
- 🎨 Modal-style feedback upon submission
- 💾 Data stored in `healthlog.txt` using clean timestamped entries
- 📁 Modular code structure for easy maintenance and scalability

---
